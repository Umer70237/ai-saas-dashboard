# AI SaaS Dashboard

A modern AI SaaS dashboard built with React.js and Tailwind CSS.

## Features

- Authentication UI flows
- Analytics dashboard
- Token usage tracking
- Cost monitoring widgets
- AI model statistics
- Dark mode interface
- Responsive design

## Tech Stack

- React.js
- Tailwind CSS
- JavaScript
- Netlify

## Live Demo

https://neuralos-ai.netlify.app


## Installation

git clone https://github.com/Umer70237/ai-saas-dashboard/

npm install

npm run dev
